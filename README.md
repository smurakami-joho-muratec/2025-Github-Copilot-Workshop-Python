# ポモドーロタイマー with ゲーミフィケーション機能

このプロジェクトは、ゲーミフィケーション要素を追加したポモドーロタイマーアプリケーションです。

## 🎮 実装されたゲーミフィケーション機能

### ✅ 経験値（XP）システム
- ポモドーロ完了時にXPを獲得
- XP累積によるレベルアップシステム
- 現在のレベルと次レベルまでの進行度表示

### ✅ 達成バッジシステム
- **初回完了**: 初めてのポモドーロ完了
- **ストリーク系**: 3日連続、1週間連続、1ヶ月連続
- **週間目標**: 週間10回、25回完了
- **月間目標**: 月間50回、100回完了
- **レベル達成**: レベル5、10到達
- **集中マスター**: 1日5時間以上の集中時間

### ✅ ストリーク表示
- 連続活動日数のカウント
- 現在のストリークと最高記録の表示
- 視覚的なストリーク表現

### ✅ 週間/月間統計
- 完了したポモドーロ数の追跡
- 集中時間の記録
- 完了率の計算
- 日別、週別、月別の詳細グラフ

### ✅ 可視化ダッシュボード
- ASCIIアートによるプログレスバー
- 横棒グラフによる週間/日別統計
- レベル進行度の可視化
- 達成バッジの進行状況

## 📁 ファイル構成

- `deliverManager.py`: メインのゲーム管理システム（ゲーミフィケーション統合済み）
- `gamification.py`: ゲーミフィケーション機能の中核システム
- `visualization.py`: 統計データの可視化モジュール
- `main.py`: 全機能デモアプリケーション
- `simple_demo.py`: シンプルなゲーミフィケーションデモ
- `main_demo.py`: より詳細なデモアプリケーション

## 🚀 実行方法

### 基本デモ
```bash
python main.py
```

### シンプルなゲーミフィケーションデモ
```bash
python simple_demo.py
```

### 可視化デモ
```bash
python visualization.py
```

### 元のデリバリーマネージャーデモ
```bash
python deliverManager.py
```

## 🎯 特徴

1. **モチベーション維持**: XPとレベルアップによる達成感
2. **継続性促進**: ストリークシステムによる継続へのインセンティブ
3. **目標設定**: 多様な達成バッジによる短期・長期目標
4. **進捗可視化**: 詳細なグラフとチャートによる成果の可視化
5. **データ永続化**: JSON形式でのデータ保存と復元

## 🧪 テスト目的

このゲーミフィケーション要素は、ポモドーロテクニックの実践において：
- ユーザーのモチベーション維持効果
- 継続利用への影響
- 集中力向上への寄与
を測定するために実装されています。

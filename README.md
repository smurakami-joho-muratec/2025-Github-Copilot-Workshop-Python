# Pomodoro Timer - 視覚的フィードバック強化版（パターンA）

ワークショップの手順：https://moulongzhang.github.io/2025-Github-Copilot-Workshop/github-copilot-workshop/#0

## 実装された機能

このプロジェクトは、視覚的没入感がユーザーの集中力に与える影響を測定するためのPomodoroタイマーアプリケーションです。

### 主要機能

✅ **円形プログレスバーのアニメーション**
- 残り時間に応じて滑らかに減少するアニメーション
- 時計回りの進行表示
- リアルタイム更新（100ms間隔）

✅ **色の変化システム**
- 時間経過に応じた色のグラデーション：青 → 黄色 → 赤
- タイマー状態による色変化（一時停止時はオレンジ、完了時は緑）
- 視覚的な緊急度の表現

✅ **背景エフェクト**
- 集中時間中のパーティクルアニメーション
- 波紋エフェクト
- 動的な背景による没入感の向上

## ファイル構成

- `main.py` - アプリケーションのエントリーポイント
- `pomodoro_timer.py` - タイマーの基本機能とロジック

- `test_headless.py` - 機能テスト（ヘッドレス環境対応）
- `test_visual_feedback.py` - 視覚機能のテスト
- `deliverManager.py` - 既存のキッチンゲーム管理システム

## 視覚効果の詳細

### 1. 円形プログレスバー
- Canvas上での円弧描画
- 進捗に応じた角度計算
- 滑らかなアニメーション更新

### 2. 色変化アルゴリズム
```python
# 0-50%: 青から黄色へ
if progress < 0.5:
    ratio = progress * 2
    r = int(74 + (255 - 74) * ratio)
    g = int(144 + (193 - 144) * ratio)
    b = int(226 + (7 - 226) * ratio)

# 50-100%: 黄色から赤へ
else:
    ratio = (progress - 0.5) * 2
    r = int(255)
    g = int(193 - (193 - 87) * ratio)
    b = int(7)
```

### 3. パーティクルシステム
- 最大20個のパーティクル
- ランダムな生成と移動
- 寿命による透明度変化

### 4. 波紋エフェクト
- 最大3個の同時波紋
- 中心からの円形拡大
- フェードアウト効果

## テスト目的

このアプリケーションは、視覚的な没入感がユーザーの集中力に与える影響を測定することを目的としています。以下の要素が集中力向上に寄与するかを検証できます：

1. 時間の視覚的表現（円形プログレス）
2. 緊急度の色による表現
3. 背景アニメーションによる没入感

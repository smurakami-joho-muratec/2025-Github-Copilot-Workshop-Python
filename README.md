# 🍅 ポモドーロタイマー - カスタマイズ版（パターンB）

ワークショップの手順：https://moulongzhang.github.io/2025-Github-Copilot-Workshop/github-copilot-workshop/#0

## 📋 プロジェクト概要

個人の好みに合わせたカスタマイズ機能を持つポモドーロタイマーアプリケーションです。  
ユーザーの継続率向上を目的とした設定オプションを提供します。

## ✨ 実装機能

### ⏰ 時間設定
- **作業時間**: 15分 / 25分 / 35分 / 45分から選択可能
- **休憩時間**: 5分 / 10分 / 15分から選択可能
- バリデーション機能により、指定されたオプション以外は設定不可

### 🎨 テーマ切り替え
- **ライトテーマ**: 白背景 + 黒文字 + 青アクセント
- **ダークテーマ**: 濃紺背景 + 白文字 + 赤アクセント  
- **フォーカステーマ**: 黒背景 + 緑文字 + 金アクセント

### 🔊 サウンド設定
- **開始音**: タイマー開始時の通知音
- **終了音**: セッション完了時のアラート音
- **tick音**: 毎秒のカウントダウン音
- 各音声は個別にオン・オフ切り替え可能

### ⚡ プリセット設定
- **標準設定**: 25分作業/5分休憩, ライトテーマ
- **短時間集中**: 15分作業/5分休憩, フォーカステーマ, tick音ON
- **長時間作業**: 45分作業/15分休憩, ダークテーマ

## 🚀 使用方法

### 基本的な実行
```bash
python3 main.py
```

### テスト実行
```bash
# 全機能テスト
python3 test_pomodoro.py

# デモンストレーション
python3 demo_pomodoro.py
```

## 📊 ユーザー継続率測定

以下の指標でユーザーの継続性を測定可能：

- ✅ 完了したポモドーロ数
- ✅ セッション継続時間
- ✅ 設定変更頻度
- ✅ 好みのテーマ使用率
- ✅ カスタム時間設定の効果
- ✅ サウンド設定の継続性への影響
